## Description

DevChat backend is a NestJS monorepo that runs as a microservice-style system:

- `gateway`: HTTP API and Socket.IO gateway.
- `auth`, `chat`, `upload`, `github`, `notifications`: Kafka microservices.
- Infrastructure: Postgres, MongoDB, Redis, Kafka, Zookeeper.

## Project setup

```bash
$ yarn install
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
